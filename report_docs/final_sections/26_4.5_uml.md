# 4.5 UML: Class and Sequence Logic

## Class-level Overview
Key classes and their responsibilities (mapped to repository modules):

- DBManager (persistence/db_manager.py): Firestore access methods (insert_entry, insert_analysis, fetch_entries_with_analysis, etc.).
- SentencePredictor (ml/inference/mood_detection/roberta/predictor.py): Loads RoBERTa, provides `predict()` and `predict_batch()`.
- SummarizationPredictor (ml/inference/summarization/bart/predictor.py): BART summarization methods.
- EmbeddingService (services/embeddings/embedding_service.py): `embed_text()` returning 384-D vectors.
- RecommendationPipeline (services/media_recommender/recommendation_pipeline.py): `get_recommendations()` orchestrates intent building, candidate fetching and ranking.
- InsightsGenerator (ml/inference/insight_generation/*/insight_analyzer.py): LLM wrapper for insight synthesis.

## Sequence Example: Entry Creation
1. Client → POST /api/v1/journal (journal_domain.create_journal_entry)
2. journal_domain.create_journal_entry → journal_entries.process_entry(user, data)
3. process_entry → DBManager.insert_entry → Firestore write
4. process_entry concurrently calls:
   - SummarizationPredictor.summarize(text)
   - SentencePredictor.predict(text)
   - EmbeddingService.embed_text(summary)
5. process_entry → DBManager.insert_analysis and DBManager.insert_embedding
6. process_entry → user_vectors update via TasteVectorService
7. Response returned to client with entry_id and analysis

## Sequence Example: Recommendation Request
1. Client → GET /api/v1/movies/recommend (media_domain.recommend_movies)
2. media_domain → RecommendationPipeline.get_recommendations(uid, media_type, filters)
3. RecommendationPipeline → intent_builder.build_intent_vector(uid, media_type)
4. intent_builder → DBManager.get_user_vectors(uid), DBManager.get_recent_embedding(uid)
5. RecommendationPipeline → media_cache fetch (`media_cache_movies`)
6. RecommendationPipeline → ranking_engine.rank_candidates(candidates, intent_vector)
7. RecommendationPipeline → response_formatter.format_recommendation_response()
8. Response returned to client
