# 5.3 Schedule

The following high-level schedule assumes a 12-week project timeline.

Weeks 1–2: Data collection and environment setup
- Configure Firestore, provision service account, provision GPU resources.
- Seed media caches and prepare small annotated datasets for mood and summarization.

Weeks 3–5: Model fine-tuning and integration
- Fine-tune RoBERTa and BART on in-domain data.
- Implement predictor wrappers and integrate into entry processing pipeline.

Weeks 6–8: Recommendation and personalization
- Implement and evaluate the recommendation pipeline, MMR and temporal decay.
- Implement user interaction logging and taste vector updates.

Weeks 9–10: Insights synthesis and evaluation
- Integrate LLM-based insights generator, design prompts and validation.
- Run user studies or human annotation for insight usefulness.

Weeks 11–12: Testing, documentation and final reporting
- Full integration testing, performance benchmarking and prepare final report and presentation materials.

This schedule is indicative and should be adapted to team size and resource availability.
