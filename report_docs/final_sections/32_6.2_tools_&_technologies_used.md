# 6.2 Tools and Technologies Used

- Programming language: Python 3.10+
- Web framework: Flask
- Database: Google Firestore via Firebase Admin SDK
- ML frameworks: PyTorch, HuggingFace Transformers, Sentence-Transformers
- Models: RoBERTa (mood detection), BART (summarization), all-mpnet-base-v2 (embeddings), Qwen2/Gemini (insights)
- Containerization: Docker, docker-compose
- Utilities: requests, numpy, scipy, logging frameworks
- Optional local LLM runtime: Ollama or HuggingFace runtime for Qwen2

This selection supports high-fidelity NLP inference, scalable service deployment and reproducible experiment workflows.
