# 3.5.2 Software Requirements (Platform Choice)

- Operating system: Linux (Ubuntu 20.04+) recommended for production; Windows/Mac supported for development.
- Python: 3.10 or higher.
- Libraries and frameworks:
  - Flask (web framework)
  - firebase-admin (Firestore and auth)
  - torch (PyTorch) and transformers (HuggingFace)
  - sentence-transformers
  - requests, numpy, scipy
- Optional runtime for local LLMs: Ollama server or HuggingFace inference endpoints.

Deployment options:
- Containerized via Docker and `docker-compose` for local deployment; orchestrated via Kubernetes for production.

These software components are used within the repository and are referenced in `requirements.txt` and Dockerfile for reproducible environments.
