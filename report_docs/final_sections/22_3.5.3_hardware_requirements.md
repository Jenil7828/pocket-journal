# 3.5.3 Hardware Requirements

Minimum development:
- CPU: quad-core (4 vCPU)
- Memory: 8 GB RAM
- Disk: 20 GB free

Recommended production:
- GPU: NVIDIA RTX-class or A100 with ≥ 16 GB VRAM for BART and local LLMs
- CPU: 8–16 vCPUs
- Memory: 32–64 GB RAM
- Network: 1 Gbps
- Storage: 100+ GB for model artifacts and media cache

Operational note: GPU instances reduce inference latency markedly for BART summarization and local LLM workloads. For cloud-based LLM usage (Gemini), CPU-only instances suffice for API orchestration.
