# 8.1 Outcomes

This section summarises principal outcomes produced by the implemented system and evaluated during integration testing and preliminary user evaluation.

1. Automated affective annotations
   - Per-entry multi-label emotion probabilities produced by the RoBERTa predictor and stored in `entry_analysis.mood`.
   - Outcome: Enables downstream mood timeline visualizations and aggregated mood statistics.

2. Abstractive summaries
   - BART-generated summaries provide concise 1–2 sentence descriptions stored in `entry_analysis.summary`.
   - Outcome: Reduced reading time and input to embedding generation.

3. Semantic embeddings for search and personalization
   - 384-D normalized embeddings stored in `journal_embeddings` and used to construct intent vectors.
   - Outcome: Improved contextual relevance in recommendations and potential for semantic search.

4. Recommendation engine
   - Context-aware recommendations returned across four media types using a hybrid ranking scheme with MMR and temporal decay.
   - Outcome: Demonstrated relevance in offline evaluation and diversity in returned lists.

5. LLM-based insights
   - Structured insights produced for date ranges and persisted in `insights`.
   - Outcome: Actionable recommendations and identified goals in structured format.

6. Reproducibility artifacts
   - Model versions, `config.yml`, and experiment logs enabling replication of results.

These outcomes demonstrate the feasibility of the integrated pipelines for personal journaling analysis and form the basis for further empirical validation.
