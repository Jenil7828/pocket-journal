# 5.4 Team Organization

Suggested team roles for project execution:

- Project Lead / Architect: Oversees system design, integration and academic reporting.
- ML Engineer(s): Responsible for fine-tuning RoBERTa and BART, embedding experiments and model deployment.
- Backend Engineer(s): Implement service layer, DB integration and API development (Flask).
- QA / Test Engineer: Design and run unit, integration, performance and security tests.
- Research Analyst: Conduct human evaluations, prepare literature review and write academic sections.

Collaboration practices:
- Weekly sprints with stand-ups and code review gates for merges.
- Shared experiment tracking for reproducibility (config.yml, logs, model versions).
