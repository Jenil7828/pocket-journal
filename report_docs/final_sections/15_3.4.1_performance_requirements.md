# 3.4.1 Performance Requirements

- API p95 latency targets:
  - Journal list and search: ≤ 200 ms
  - Recommendation (10 items): ≤ 500 ms
  - Insights generation (cloud LLM): ≤ 2 s

- ML inference:
  - RoBERTa: <500 ms per entry on GPU
  - BART summarization: <1 s per entry on GPU
  - Embeddings: <100 ms per summary on GPU

- Throughput:
  - System to sustain ≥ 100 RPS in steady-state with horizontal scaling.

- Batch sizes and concurrency:
  - Embedding batch size default 32, inference batch default 16 for summarization (configurable in `config.yml`).

Performance verification: use load testing suites and measure p50/p95/p99; include hardware spec in benchmarks.
