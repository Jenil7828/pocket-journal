## 🎁 BONUS RESOURCES

### Statistical Test Calculator
If you need to compute p-values:

```python
# Python scipy one-liner:
from scipy import stats
t_stat, p_value = stats.ttest_rel(condition_a, condition_b)
print(f"t={t_stat:.2f}, p={p_value:.4f}")

# Mark as significant (✓) if p<0.05
# Mark as marginal (◆) if 0.05 ≤ p < 0.10
# Mark as ns (✗) if p≥0.10
```

### Effect Size Guide
When reporting differences:

```
Small effect: d = 0.2  (5-10% difference)
Medium effect: d = 0.5  (10-20% difference)
Large effect: d = 0.8+  (20%+ difference)

Your intent vector: 5% vs. 0%
```
