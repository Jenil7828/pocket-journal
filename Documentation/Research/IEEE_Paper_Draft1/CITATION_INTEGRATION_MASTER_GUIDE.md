## 📞 IF YOU GET STUCK

**Problem:** Can't find author names in PDF
→ Check: Title page, header, or first paragraph

**Problem:** No DOI visible
→ Solution: Mark as "doi: Not available" (still valid in IEEE)

**Problem:** Multiple authors - how many to list?
→ Rule: List first 3, then use "et al." if more than 3

**Problem:** Conference name unclear
→ Check: Usually says "Proceedings of [Conference Name]"

**Problem:** Page numbers not visible
→ Solution: Use page numbers shown in PDF viewer
