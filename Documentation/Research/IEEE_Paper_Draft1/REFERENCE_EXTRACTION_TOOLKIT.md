## 📐 IEEE FORMAT RULES

### Journal Article:
```
[#] Initial(s). Surname, Initial(s). Surname, et al., "Title of article," 
Title of Journal, vol. #, no. #, pp. page range, Abbrev. Month Year, 
doi: DOI.

Example:
[1] Y. Liu, M. Ott, N. Goyal, et al., "RoBERTa: A robustly optimized BERT 
pretraining approach," arXiv preprint, 2019, arXiv:1907.11692.
```

### Conference Proceeding:
```
[#] Initial(s). Surname, et al., "Title of paper," in Proceedings of Conf., 
City, State/Country, Month Day–Day, Year, pp. range.

Example:
[2] D. Sculley, et al., "Hidden technical debt in machine learning systems," 
in Advances in Neural Information Processing Systems (NeurIPS), Montreal, QC, 
Canada, Dec. 2015, pp. 2503–2511.
```

### Book:
```
[#] Initial(s). Surname, Initial(s). Surname, Title of Book. Publisher, Year, 
pp. page range.

Example:
[3] G. Adomavicius and A. Tuzhilin, Context-aware Recommender Systems. 
Springer, 2015, pp. 191–226.
```
