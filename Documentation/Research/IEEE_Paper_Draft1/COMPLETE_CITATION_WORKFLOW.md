## 💡 PRO TIPS

**Tip 1:** When extracting, copy text directly from PDF (Ctrl+C) to avoid typos

**Tip 2:** If DOI unavailable, mark as "doi: Not available" - still valid in IEEE format

**Tip 3:** For conference proceedings, include conference location if available:
```
[#] Author(s), "Title," in Proc. Conf. Name, City, Country, 
Month Year, pp. ##–##.
```

**Tip 4:** For many authors (>3), use "et al." after 3rd author:
```
[#] A. Smith, B. Jones, C. Brown, et al., "Title," ...
```

**Tip 5:** Keep a running list in Excel/Google Sheets as you extract to prevent duplicates
