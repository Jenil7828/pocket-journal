📓 Pocket Journal — Mood-Based Recommendations API

Welcome! Use the following endpoints to analyze journal entries and get personalized recommendations for movies, songs, and books.

📝 Journal Entry Processing

POST /process_entry

Request JSON:

{
  "entry_text": "Your journal text here..."
}

Response: entry_id, summary, mood probabilities

🔍 Generate Insights

POST /generate_insights

Request JSON:

{
  "start_date": "2025-09-01",  // optional 
  "end_date": "2025-09-30"     // optional
}

Response: Extracted insights (goals, progress, challenges, etc.)

✏️ Update Entries

  • Update single entry: PUT /entries/{entry_id}
    Request body: {"entry_text": "new text", "regenerate_analysis": true}
    Updates the journal entry and regenerates its analysis
  • Reanalyze entry: POST /entries/{entry_id}/reanalyze
    Regenerates analysis for an existing entry without changing the text

📖 Get Entries

  • Get all entries: GET /entries
    Query params: ?start_date=2025-01-01&end_date=2025-01-31&mood=happy&search=keyword&limit=50&offset=0
  • Get single entry: GET /entries/{entry_id}
    Returns entry with analysis
  • Get entry analysis: GET /entries/{entry_id}/analysis
    Returns only mood analysis for the entry

🗑️ Delete Entries

  • Delete single entry: DELETE /entries/{entry_id}
    Deletes the journal entry and its associated analysis
  • Delete multiple entries: DELETE /entries/batch
    Request body: {"entry_ids": ["id1", "id2", ...]}

💡 Insights Management

  • Get all insights: GET /insights
    Query params: ?limit=50&offset=0
  • Get specific insight: GET /insights/{insight_id}
    Returns detailed insight information
  • Delete insight: DELETE /insights/{insight_id}
    Deletes insight and its mappings

📊 Analytics & Statistics

  • User statistics: GET /stats
    Returns total entries, insights, mood distribution, recent activity
  • Mood trends: GET /mood-trends
    Query params: ?days=30 - Returns mood trends over time

🔧 Utility & Export

  • Health check: GET /health
    Returns API health status and service connectivity
  • Export data: GET /export
    Query params: ?start_date=2025-01-01&end_date=2025-01-31&format=json
    Supports JSON and CSV formats

🎬 Movies

  • Get movies by mood: /api/recommend?mood=happy
  • Search movies (typo-tolerant): /api/search?movie=Incepton

🎵 Songs

  • Get songs by mood: /api/songs?mood=happy&language=both&limit=5
    - mood: happy, sad, chill, energetic, romantic
    - language: english, hindi, both (default = both)
    - limit: number of songs (default = 10)
  • Search songs or artists: /api/search_song?q=arjit sngh&type=artist&limit=10
    - q: Song or artist name (typo-tolerant)
    - type: track or artist (default = track)
    - limit: number of results (default = 10)

📚 Books

  • Get books by emotion: /api/books?emotion=happy&limit=5
    Emotions: happy, sad, angry, romantic, stressed, bored
  • Search books: /api/search_books?query=harry poter&type=both&limit=5
    - query: book title or author
    - type: title, author, or both (default = both)
    - limit: number of results (default = 10)
